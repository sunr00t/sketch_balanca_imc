- Arduíno Uno;
- Protoboard;
- Balança Digital c/ 04 células de carga;
- Display LCD 16x2;
- Módulo I2c Interface Serial p/ LCD(16x02);
- Módulo Conversor HX711 (p/ células de carga);
- Pushbotton + Resistor 10k;
- Sensor Ultrassônico (HC-SR04);
